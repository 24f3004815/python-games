## 🎮 What is this?

This is a classic terminal-based **Rock Paper Scissors** game. You play against the computer (RNGesus). It's fast, it's fun, and it has a *personality*.

## 🚀 How to Play

Navigate to the directory and run it in your terminal:

```bash
python app.py
```

The game will ask you for your choice. 

```
Enter the number : 
   1. ROCK
   2. PAPER
   3. SCISSOR
```

Just type `1`, `2`, or `3` and the battle begins! 💥

## ✨ Features

*   🏆 **Live Scoreboard**: Win, Lose, and Draw counters are updated in real-time.
*   🔁 **Endless Rounds**: Keep playing as long as you want.
*   🧠 **Randomized AI**: Each of the computer's choices is truly random.
*   🎭 **Personality**: The game is lively, with some humorous responses to wrong inputs.

## 📝 Game Rules

*   **Rock** beats **Scissors**
*   **Scissors** beats **Paper**
*   **Paper** beats **Rock**
*   Same choice? It's a **Draw**!

## 📦 Requirements

*   Python 3.x
*   No external dependencies required!
