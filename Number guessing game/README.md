# 🎮 Number Guessing Game

A simple command-line Python game where the player tries to guess a randomly generated number within a limited number of attempts. This project was built to practice fundamental Python programming concepts such as loops, conditionals, functions, user input, and the `random` module.

---

## ✨ Features

* Random number generation
* User input validation
* Too High / Too Low hints
* Attempt counter
* Win/Lose conditions
* Play again option (if implemented)

---

## 🛠️ Technologies Used

* Python 3
* Built-in `random` module


---

## 🎯 How to Play

1. Run the program.
2. The computer randomly selects a secret number.
3. Enter your guess.
4. The game will tell you whether your guess is **Too High** or **Too Low**.
5. Continue guessing until you find the correct number.
6. Your total attempts will be displayed when you win.
